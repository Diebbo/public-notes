# Implementazione

- **Time:** $O(n^2)$
- **Memory:** $O(n^2)$

```python
class Solution:
    def largestDivisibleSubset(self, nums: List[int]) -> List[int]:
        nums = sorted(nums)
        n = len(nums)
        dp = [[j] for j in nums]
        m = 1
        ind = 0
        for i in range(1, n):
            for j in range(i-1, -1, -1):
                if nums[i] % nums[j] == 0:
                    if len(dp[j]) + 1 > len(dp[i]):
                        dp[i] = dp[j] + [nums[i]]
                        
                        ind,m = (ind, m) if max(m, len(dp[i])) == m else (i, len(dp[i]))
        print(dp)
        return dp[ind]
```
