# Implementazione

- **Time:** $O(n)$
- **Memory:** $O(1)$

```python
class Solution:
    def tribonacci(self, n: int) -> int:
        if n == 0 or n == 1:
            return n
        if n == 2:
            return 1
        n-=2
        n1, n2, n3 = (0, 1, 1)
        while(n > 0):
            n1, n2, n3 = (n2, n3, n1+n2+n3)
            n-=1
        return n3
            
```

>[!Important] **Intuizione**
> - letteralmente fibonacci
