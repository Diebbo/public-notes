# Implementazione

- **Time:** $O(n\log n)$
- **Memory:** $O(1)$

```python
class Solution:
    def largestPerimeter(self, nums: List[int]) -> int:
        N = len(nums)
        nums = sorted(nums)
        s = sum(nums)

        for i in range(N-1, 1, -1):
            s -= nums[i]
            if s > nums[i]:
                return s + nums[i]
            
    
        return -1
```

>[!Important] **Intuizione**
> - settimana greedy, implementazione molto figa e pulita per un problema complesso
