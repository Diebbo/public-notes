# Implementazione

- **Time:** $O(n*m)$
- **Memory:** $O(1)$

```python
class Solution:
    def islandPerimeter(self, grid: List[List[int]]) -> int:
        N = len(grid)
        M = len(grid[0])
        perimitier = 0
        for i in range(N):
            for j in range(M):
                if grid[i][j] == 1:
                    perimitier += 1 - grid[i-1][j] if i > 0 else 1
                    perimitier += 1 - grid[i+1][j] if i < N-1 else 1
                    perimitier += 1 - grid[i][j-1] if j > 0 else 1
                    perimitier += 1 - grid[i][j+1] if j < M-1 else 1
        return perimitier
        
```
